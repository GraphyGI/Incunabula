Tu Guía de Mantenimiento

Recomendaciones sencillas para cuidar tu tienda

1. SEGURIDAD ACCESO

Activar WP 2FA

Instalar el plugin gratuito WP 2FA. Configura la autenticación obligatoria en dos pasos para Administradores y Gestores. Protege el sitio de ataques de fuerza bruta requiriendo un código móvil temporal desde Google Authenticator.

2. OPTIMIZACIÓN

Definir Estado de YayMail

Decidir si personalizar las 19 plantillas de correos inactivas en YayMail → Email Templates o desactivar el plugin. Tenerlo activo sin usar consume recursos del servidor en cada compra sin aportar valor visual.

3. SEGURIDAD DNS

Escalar Registro DMARC

Tras 4 semanas de envíos estables con Brevo, cambia el registro DNS DMARC en tu panel de Hostinger de p=none a p=quarantine. Bloqueará eficazmente cualquier correo fraudulento que intente suplantar tu dominio.

4. BLINDAJE SERVER

Cabeceras HTTP en .htaccess

Insertar reglas de protección HTTP (HSTS, CSP y X-Frame-Options) en el archivo .htaccess. Esto blindará la experiencia de tus clientes evitando secuestros de clics (clickjacking) o inyección de scripts externos.

5. PREVENCIÓN

Simular Copia de Seguridad

Programar una simulación mensual en Hostinger. Consiste en restaurar el backup automático más reciente en un subdominio o base de datos de pruebas para validar la integridad absoluta del sistema de prevención ante caídas.

6. QA FINAL

Compra Real de Validación

Ejecutar un pedido real de bajo costo con Wompi para asegurar que la conexión de producción responde, la pasarela descuenta correctamente y se dispara el correo de confirmación de Brevo sin retrasos.
